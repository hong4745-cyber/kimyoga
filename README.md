# ✨ 프로젝트 소개

기존 홈페이지는 오래된 UI와 복잡한 정보 구조로 인해 원하는 정보를 빠르게 찾기 어려웠으며, 모바일 환경에서도 사용성이 다소 제한적이었습니다.

본 프로젝트는 이러한 문제를 개선하기 위해 **정보 구조(IA)를 재설계**하고, **반응형 UI/UX**, **Firebase 기반 회원 기능**, **다양한 인터랙션**을 적용하여 사용자 경험을 향상시키는 것을 목표로 진행한 홈페이지 리뉴얼 프로젝트입니다.

# 📖 프로젝트 개요

기존 요가·필라테스 교육기관 홈페이지를 사용자 중심으로 리뉴얼한 프로젝트입니다.

반응형 레이아웃과 다양한 인터랙션을 적용하여 사용자 경험을 개선했으며, Firebase를 활용한 회원 인증 및 커뮤니티 기능, NAVER Map API와 소셜 로그인 연동 등 실제 서비스에 필요한 기능을 구현했습니다.

또한 디자인부터 퍼블리싱, 프론트엔드 개발까지 전 과정을 직접 진행하며 실무와 유사한 개발 프로세스를 경험했습니다.
